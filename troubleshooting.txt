Troubleshooting:

1) Your chromedriver executable version needs to be the same as the one that you have installed. (session not created: This version of ChromeDriver only supports Chrome version)
You can solve this issue by following the steps laid out here:
 "https://stackoverflow.com/questions/65194694/session-not-created-this-version-of-chromedriver-only-supports-chrome-version-8

2) Chromedriver executable needs to be in path error:
You may have your chromedriver in the same directory as the scripts for wardrobe wizard, sometimes that still doesn't work super well and you need to pass the fully qualified path.

3) Unverified developer error (Mac OS):
You may find that there is an unexpected quit error and the chromedriver never opens up, this is likely because you haven't enabled the necessary permissions for the chromedriver. You can solve this by following these instructions:
https://stackoverflow.com/questions/60362018/macos-catalinav-10-15-3-error-chromedriver-cannot-be-opened-because-the-de

4) Running "flask run" may not work, try:
    Re-setting '''export FLASK_APP...'''
    Re-run '''flask run'''
    Failing that, run the command '''python3 -m flask run'''
    That should work
